# Arik Mukherjee - Portfolio Website

A modern, responsive personal portfolio website showcasing my professional journey as a Computer Science student and Web Developer.

## 🌟 About

This is my personal portfolio website built with vanilla HTML5 and CSS3. It highlights my educational background, technical skills, and professional experience as a Computer Science student and aspiring web developer.

## ✨ Features

- **Modern Design**: Clean, gradient-based UI with contemporary styling
- **Responsive Layout**: Fully responsive design that works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Hover effects and transitions for enhanced user experience
- **Semantic HTML**: Properly structured HTML5 markup for accessibility and SEO
- **CSS Variables**: Efficient styling with CSS custom properties for easy theme management
- **Professional Sections**:
  - Navigation bar with sticky positioning
  - Hero section with profile image and introduction
  - About section with skill cards
  - Education timeline
  - Experience information
  - Social media links and contact information

## 🎨 Design Highlights

- **Color Scheme**: Purple gradient primary (#667eea to #764ba2), dark secondary (#1e293b), accent pink (#ec4899)
- **Layout System**: CSS Grid and Flexbox for responsive multi-column layouts
- **Breakpoint**: Mobile responsiveness at 768px
- **Typography**: Clean, readable font hierarchy with proper contrast
- **Effects**: Gradient backgrounds, box shadows, backdrop filters, and smooth transitions

## 🚀 Quick Start

1. Open `index.html` in your web browser
2. Navigate through sections using the sticky navigation bar
3. View the portfolio on different screen sizes to see responsive design in action
